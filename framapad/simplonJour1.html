Exercice Fizzbuzz :
    
    renvoie fizz quand multiple de 3
    renvoie buzz quand multiple de 5
    fizzbuzz quand multiple de 3 et 5
    
Comment faire ? --> on créé un fichier test qui va tester d'autres fichiers

Je vous propose cette solution (sans garantie, mais ça fonctionne) :  
    
    Algorithme pour la classe fizzbuzz : 
    
    1  
2 class Fizzbuzz 3 def fb(i) 4 str="" 5 if (i%3==0) 6 str="Fizz" 7 end 8 if (i%5==0) 9 str=str+"Buzz" 10 end 11 if (i%3!=0&&i%5!=0) 12 str=str+i.to_s 13 end 14 return str 15 end 16 end     
    Code de test (fizzbuzz_test.rb)
1 require 'minitest/autorun' 2 require './fizzbuzz.rb' 3 4 class FizzbuzzTest<Minitest::Unit::TestCase 5 def setup 6 @fizzbuzz=Fizzbuzz.new 7 end 8 def test_donne_1 9 assert_equal "1",@fizzbuzz.fb(1) 10 end 11 def test_donne2 12 assert_equal "Fizz",@fizzbuzz.fb(3) 13 end 14 def test_donne3 15 assert_equal "FizzBuzz",@fizzbuzz.fb(15) 16 end 17 def test_donne4 18 assert_equal "11", @fizzbuzz.fb(11) 19 end 20 def test_donne5 21 assert_equal "Fizz",@fizzbuzz.fb(18) 22 end 23 def test_donne6 24 assert_equal "Buzz",@fizzbuzz.fb(20) 25 end 26 end
